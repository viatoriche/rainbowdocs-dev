Оформление документов для занесения в базу данных openchain:

Создается простой документ в формате OpenDocument (openoffice.org)
Изменяемая часть выделяется тегами {% название_тега|описание тега %}

Пример:

{% FIO|ФИО %}

У каждого документа должно быть название, оформленное специальным выделением:
{{ begin_title }}Название документа{{ end_title }} так можно выделить любую область в тексте
которая будет являться названием.
Также у каждого документа должна быть область для номера.
Она указывается специальным тегом: {{ number }}
